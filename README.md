# Leland Origami Club Website

Welcome to the **Leland Origami Club** website repository! 
This project showcases the club’s activities, gallery of member creations, and a way for new members to join. 

## Development

This project now uses Svelte with Vite.

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Create production build: `npm run build`
4. Preview production build: `npm run preview`
